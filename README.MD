# SixSeries

A streaming application developed by Gabriel and Gerardas.

# Overview

SixSeries is a streaming application designed to provide users with a smooth and engaging experience for discovering and watching their favorite series and movies. The app focuses on simplicity, speed, and a clean user interface.

# Features

Browse and stream series and movies

Search by title, genre, or actor

Personalized recommendations

Save favorites and create watchlists

User login and registration system
